Migrate Ceylon Repositories

Make sure to create an empty repository named ceylon/ceylon-obr-test with the following team rights:

  • obr-migration - Admin
  • Owners - Read
  • ceylon-core - Read
  • spec-typechecker - Read
  • email-receiver - Read
  • ceylon/ceylon-ceylon-organization-member-no-right - Read

And don't forget to uncomment the line at the start of the migrate.js file when doing the real migration!
